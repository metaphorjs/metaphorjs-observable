class Observable

  • @version 1.2
  • @author Ivan Kuindzhi
  • @link https://github.com/kuindji/metaphorjs-observable

A javascript event system implementing two patterns - observable and collector.

Observable:

var o = new Observable;
o.on("event", function(x, y, z){ console.log([x, y, z]) });
o.trigger("event", 1, 2, 3); // [1, 2, 3]

Collector:

var o = new Observable;
o.createEvent("collectStuff", "all");
o.on("collectStuff", function(){ return 1; });
o.on("collectStuff", function(){ return 2; });
var results = o.trigger("collectStuff"); // [1, 2]

Methods

  • createEvent (name, returnResult, autoTrigger, triggerFilter, filterContext)  -> ObservableEvent

    • @access public

    You don't have to call this function unless you want to pass params other than event name. Normally, events are created automatically.

    • required
      string name

      Event name

    • boolean | string returnResult

      false -- return first 'false' result and stop calling listeners after that
      "all" -- return all results as array
      "merge" -- merge all results into one array (each result must be array)
      "first" -- return result of the first handler (next listener will not be called)
      "nonempty" -- return first nonempty result
      "last" -- return result of the last handler (all listeners will be called)

    • boolean autoTrigger

      once triggered, all future subscribers will be automatically called with last trigger params

      var o = new Observable;
o.createEvent("auto", null, true);
// trigger first
o.trigger("auto", 1, 2);
// subscribe later
o.on("auto", function(a, b){
    console.log(a, b); // immediately logs 1, 2
});
    • function triggerFilter (listener, arguments)  -> boolean

      This function will be called each time event is triggered. Return false to skip listener.

      var o = new Observable;
o.createEvent("filtered", null, false, function(l, args){
    if (l.always) {
        return true;
    }
    if (l.param == args[0]) {
        return true;
    }
    return false;
});

o.on("filtered", function(){
    console.log("always")
}, null, {
    always: true
});

o.on("filtered", function(){
    console.log("param");
}, null, {
    param: 1
});


o.trigger("filtered", 2); // "always"
o.trigger("filtered", 1); // "always", "param"
      • object listener

        This object contains all information about the listener, including all data you provided in options while subscribing to the event.

      • array arguments
    • object filterContext

      triggerFilter's context

mixin Observable

Mixin adds observable features to the host object. It adds 'callback' option to the host config. See $beforeInit. Mixin is designed for MetaphorJs class system.

var MyClass = defineClass({
    $class: "MyClass",
    $mixins: ["mixin.Observable"],
    $init: function() {
        this.trigger("someEvent", 123);
    },

    doSomeAction: function() {
        // does some action
        this.trigger("anotherEvent", 234);
    }
});

var ParentClass = defineClass({
    $init: function() {
        this.my = new MyClass({
            callback: {
                context: this,
                someEvent: this.onSomeEvent
            }
        });
        this.my.on("anotherEvent", this.onAnotherEvent, this);
        this.my.doSomeAction();
    },

    onSomeEvent: function(value) {
        // value == 123
    },

    onAnotherEvent: function(value) {
        // value == 234
    }
});

var p = new ParentClass;
